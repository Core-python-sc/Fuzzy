# Scalar Cardinality & Relative Cardinality in Fuzzy Sets

This project explains the fundamental concepts of **Scalar Cardinality** and **Relative Cardinality** in fuzzy set theory with clear mathematical representation.

## 🔹 Scalar Cardinality

Scalar cardinality measures the total membership of elements in a fuzzy set.

### Mathematical Representation:

|A| = Σ μA(xᵢ)

It is calculated by summing all membership values of the elements in the set.

---

## 🔹 Relative Cardinality

Relative cardinality represents the average membership value of the fuzzy set.

### Mathematical Representation:

R(A) = (1/n) Σ μA(xᵢ)

It is obtained by dividing the scalar cardinality by the total number of elements.

---

## 📌 Key Difference

| Concept              | Meaning                  |
| -------------------- | ------------------------ |
| Scalar Cardinality   | Total membership value   |
| Relative Cardinality | Average membership value |
